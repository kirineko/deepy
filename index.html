Syntax-highlighted changes

diff 里的新增和删除代码会根据文件路径推断语言，并用 Rich/Pygments 渲染 token 颜色，同时保留 old/new 行号和 diff 背景。

04

Dark and light theme coverage

变更预览覆盖 dark/light palette，宽字符 padding、超大 write 预览和实时流式 工具输出都有回归测试保护。

为 DeepSeek 编程会话而设计

Deepy 是一个 Python CLI，通过 OpenAI Agents SDK 调用 DeepSeek。它围绕 thinking 模式、长上下文、上下文缓存和可读的终端 UI 进行优化，让 Agent 的 工作过程清晰可见。

理解项目 自动加载项目规则、skills、会话历史和当前工作目录。
工具驱动 读取文件、编辑代码、运行命令、搜索资料，并抓取指定 URL。
终端体验 渲染 Markdown、thinking、diff、每轮 usage 和上下文窗口状态。
长上下文可靠性 用 pending token accounting、reserved budget 和可恢复归档保护长会话。

终端优先的工作流

Deepy 把对话、工具调用、代码修改和验证输出放在同一个终端上下文里。

> code

编写代码并运行验证

让 Deepy 实现功能、补充测试、运行项目命令并总结结果。diff 会直接在终端中 渲染，方便你快速审查改动。

> explain

把执行结果整理成说明

Deepy 可以把命令输出和项目文件整理成可读的 Markdown，包括项目结构、 实现细节和测试覆盖。

> research

检索并抓取来源页面

WebSearch 用于查找最新资料；WebFetch 可以读取指定 URL，在需要精确网页 内容时给出有来源依据的回答。

安装与维护

推荐用 uv 安装 Deepy。国内网络环境可以先配置清华 PyPI 镜像，再安装、 升级或卸载命令行工具。

1. 准备 uv

macOS / Linux

curl -LsSf https://astral.sh/uv/install.sh | sh

Windows PowerShell

powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

2. 配置清华镜像（可选）

Linux / macOS: ~/.config/uv/uv.toml

Windows: %AppData%\uv\uv.toml

[[index]]
url = "https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple/"
default = true

3. 安装并启动 Deepy

uv tool install deepy-cli

# 进入你的项目目录
cd your-project
deepy

4. 升级 Deepy

uv tool upgrade deepy-cli

5. 卸载 Deepy

uv tool uninstall deepy-cli
