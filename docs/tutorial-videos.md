# Deepy Video Tutorials

This page collects the Deepy tutorial video series on Bilibili. Use it when you
prefer learning by watching a complete workflow before reading reference docs.

Season page:
[Deepy tutorial series](https://space.bilibili.com/560507/lists/8171057)

## How To Use These Videos

- Start with the Quick Start in [README.md](../README.md) if you only need to
  install and run Deepy.
- Watch the first videos to build the basic workflow: launch Deepy, ask it to
  inspect a project, edit files, and run validation.
- Use the later videos when you want examples for MCP, skills, subagents, and
  larger application tasks.
- Use the reference docs when you need exact configuration fields or safety
  boundaries:
  - [MCP configuration](mcp.md)
  - [Subagents](subagents.md)
  - [Deepy UI and TUI](deepy-ui-and-tui.md)

The table uses the original Bilibili titles. Links intentionally omit tracking
query parameters.
