# Deepy 教学视频

本文整理 Deepy 在 B 站的教学视频合集。如果你更习惯先看完整操作流程，再查文字文档，
可以从这里开始。

合集入口：
[Deepy 教学视频合集](https://space.bilibili.com/560507/lists/8171057)

## 如何使用这些视频

- 如果只想安装并跑起来，先看 [README.zh-CN.md](../README.zh-CN.md) 的快速开始。
- 前几集适合建立基础工作流：启动 Deepy、让它理解项目、修改文件、运行验证。
- 后续视频适合查看 MCP、skills、subagents 和更大应用任务的示例。
- 需要精确配置字段或安全边界时，以文字参考文档为准：
  - [MCP 配置](mcp.zh-CN.md)
  - [Subagents](subagents.zh-CN.md)
  - [Deepy UI 与 Deepy TUI](deepy-ui-and-tui.zh-CN.md)
